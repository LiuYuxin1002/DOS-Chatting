# DOS-Chatting
1.简介：
  一个基于dos命令行窗口的小程序，具有多用户聊天（但不支持群聊）、实时显示在线人数、随时切换自己的聊天对象、上下线通知等功能。
2.结构：
  基于Java的TCP通讯协议。
  程序分为两部分：Server.java 和 Client.java。
  服务端提供登录、命令、信息转发等功能；
  客户端接收服务端消息打印到屏幕上，并可将自己的消息发送给服务端，令其转发给自己的好友。
